# 🌙 灵感穹顶 - 部署教程

这是一份傻瓜式教程，教你如何把"灵感穹顶"部署到网上，然后在华为手机上像 App 一样使用。

---

## 📋 你需要准备的

1. **一个 GitHub 账号**（免费）
   - 如果没有，去 https://github.com 注册一个

2. **一个 Vercel 账号**（免费）
   - 直接用 GitHub 账号登录即可

---

## 🚀 第一步：上传代码到 GitHub

### 方法 A：网页操作（最简单）

1. 登录 GitHub，点击右上角 **+** → **New repository**

2. 填写信息：
   - Repository name: `inspiration-vault`
   - 选择 **Public**
   - 点击 **Create repository**

3. 在新页面，点击 **uploading an existing file**

4. 把我给你的 **inspiration-vault-pwa** 文件夹里的所有文件拖进去：
   ```
   📁 inspiration-vault-pwa
   ├── 📁 public/
   │   ├── icon-192.png
   │   ├── icon-512.png
   │   ├── apple-touch-icon.png
   │   └── favicon.ico
   ├── 📁 src/
   │   ├── App.jsx
   │   └── main.jsx
   ├── index.html
   ├── package.json
   └── vite.config.js
   ```

5. 点击 **Commit changes**

### 方法 B：使用 Git 命令（如果你熟悉的话）

```bash
cd inspiration-vault-pwa
git init
git add .
git commit -m "初始提交"
git remote add origin https://github.com/你的用户名/inspiration-vault.git
git push -u origin main
```

---

## 🌐 第二步：部署到 Vercel

1. 打开 https://vercel.com

2. 点击 **Sign Up** → 选择 **Continue with GitHub**

3. 授权 Vercel 访问你的 GitHub

4. 进入 Dashboard 后，点击 **Add New...** → **Project**

5. 在列表中找到 `inspiration-vault`，点击 **Import**

6. 配置页面保持默认，直接点击 **Deploy**

7. 等待 1-2 分钟，部署完成！

8. 你会得到一个网址，类似：
   ```
   https://inspiration-vault-xxx.vercel.app
   ```

---

## 📱 第三步：在华为手机上安装

### 使用华为浏览器

1. 打开 **华为浏览器**

2. 输入你的 Vercel 网址，比如：
   ```
   https://inspiration-vault-xxx.vercel.app
   ```

3. 等页面完全加载后，点击浏览器菜单（通常是右下角三个点）

4. 找到 **添加到主屏幕** 或 **添加到桌面**

5. 确认添加

6. 回到手机桌面，你会看到一个"灵感穹顶"的图标！

### 使用 Chrome 浏览器

1. 打开 Chrome

2. 输入你的网址

3. 点击右上角三个点 → **添加到主屏幕**

4. 点击 **添加**

## ❓ 常见问题

### Q: 添加到主屏幕选项在哪？
A: 不同手机浏览器位置不同，一般在菜单里找"添加到桌面"、"添加快捷方式"或类似选项

### Q: 数据会丢失吗？
A: 数据保存在手机浏览器的本地存储中，只要不清除浏览器数据就不会丢失

### Q: 可以多设备同步吗？
A: 目前是单机版，数据不同步。如果需要同步功能，以后可以加云存储

### Q: 华为手机可以安装 APK 吗？
A: 可以！但 PWA 方式更简单。如果你以后想要 APK，我可以帮你打包
